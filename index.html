Hola como estas!

Soy una desarrolladora de software orientada a construir soluciones eficientes, lógicas y escalables. Cuento con una sólida base técnica centrada en lenguajes como C# y C++, complementada con el modelado de bases de datos relacionales en SQL Server y el análisis con Power BI.
